# CVai — AI CV Builder

Paste a job description, get a tailored CV, preview it, and download as PDF.

## Stack

- React + Vite (JavaScript)
- React Router
- Tailwind CSS
- Supabase (auth)
- Google Gemini API
- Framer Motion

## Setup

1. Copy `.env.example` to `.env` and fill in your keys.
2. `npm install`
3. `npm run dev`

## Supabase

Enable Email auth in your Supabase project dashboard.
